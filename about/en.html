About

AB‑DOC is a web-application, developed by ERP‑LAB. The project was started in the summer of 2017. In January 2018 we released public beta‑version.

AB‑DOC allows you to create documents online and organize them in tree structure. You can store your thoughts, information, you found in Internet, or whatever. Application is serverless, based on Amazon Web Services. All users' data is stored in Amazon S3 storage.

AB‑DOC is free and current functionality will always be free. All user documents are public. Every user has 1Gb storage limit.

By the summer 2018 we are planning to add pay accounts (Pro) for 6$ per month. Pro-accounts will have increased space limit (100Gb) and ability to make private docs or whole branches.
